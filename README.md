# FinPt

경제 뉴스와 학습을 결합한 인터랙티브 학습 플랫폼

## 🚀 주요 기능

### 1. 오늘의 이슈 요약
- 주요 경제 뉴스를 간단히 요약하여 제공
- 플립 카드 UI로 직관적인 정보 전달
- 전체 뉴스 보기 기능으로 상세 정보 확인 가능

### 2. 맞춤형 학습
- 금융시장 기초부터 투자 전략까지 다양한 주제 학습
- 개인별 학습 진도 추적 시스템
- 단계별 학습으로 체계적인 지식 습득

### 3. 리그 & 챌린지
- 주간 리그 참여로 다른 사용자들과 경쟁
- 일일 챌린지를 통한 지속적인 학습 동기 부여
- 출석 체크 및 보상 시스템으로 학습 의욕 고취

### 4. 오늘의 이슈와 관련된 숏폼
- MZ들에게 적합한 숏폼형식으로 금융정보들에 친숙하게 접근
- 기존 숏폼과 같은 디자인으로 친숙하게 사용
- 재미뿐만 아니라 지식 습득

### 5. 금융 Chatbot
- 실시간 정보들을 바탕으로 답변
- 가독성과 쉬운 단어들로 설명
- 앱을 이용하는 도중 언제든지 동시에 띄워 간편하게 사용

## 🛠 기술 스택

- **프론트엔드**: Next.js 14, TypeScript
- **상태 관리**: Recoil
- **UI/UX**: Tailwind CSS, shadcn/ui
- **데이터 관리**: TanStack Query

## 🚀 시작하기

### 설치 방법

0. API_KEY
   Need your own API_KEY

1. Dependency:
   ```
   cd {project_folder}
   npm install
   ```

2. Run server:
   ```
   npm run dev
   ```

3. 브라우저에서 `http://localhost:3000` 접속
